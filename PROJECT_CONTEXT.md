# ЯД Оптимизатор — Контекст проекта

## Идея

Сервис для автоматического управления рекламой в Яндекс.Директ.
Пока используется только владельцем (Владимир), монетизация не рассматривается.

## Основная концепция

**Единица = Сайт** (rogikopyta.ru), а не отдельная кампания.
Под каждым сайтом — несколько кампаний: Поиск, РСЯ, Смарт-баннеры и т.д.

### Флоу пользователя:
1. Вводит URL сайта
2. Сервис анализирует сайт + запрашивает Wordstat (реальные данные, без галлюцинаций)
3. Получает рекомендации: ключевики, минус-фразы, структуру кампаний, креативы
4. Принимает или отклоняет каждую рекомендацию точечно
5. Авторизуется через Яндекс → кампании создаются в Директе
6. Раз в неделю (или по расписанию) сервис дёргает статистику и делает отчёт

### Отчёт включает:
- Обзор метрик за период (сравнение с предыдущим)
- Разбивка по сайтам и кампаниям
- Аудит сайта (скорость, мобилка, контент) — особенно первый отчёт
- Рекомендации в конце с обоснованием и данными
- Изменения бюджетов (редактируемые прямо в сервисе перед отправкой в Директ)
- Кнопка "Отправить изменения в Директ"

### Состояние "нет Директа/Метрики":
- Отдельный UI с инструкциями как подключить
- Ссылки на документацию
- Возможность работать в демо-режиме

## Планируемые интеграции (API)

| Сервис | Назначение | Токен |
|---|---|---|
| Яндекс.Директ | Управление кампаниями, статистика | OAuth токен |
| Яндекс.Метрика | Данные о конверсиях | Тот же OAuth |
| Яндекс Wordstat | Частотность ключевых слов | Через токен Директа |
| YandexGPT | Генерация текстов объявлений | Яндекс Облако API key + Folder ID |
| YandexART | Генерация изображений для РСЯ | Тот же Яндекс Облако |
| GigaChat (Сбер) | Альтернативная генерация текстов | Client ID + Secret |
| Claude API | Анализ и гипотезы | Anthropic API key |

> Архитектура: любую нейросеть можно включить/выключить в настройках.
> Все токены хранятся в localStorage (только для локальной разработки).

## Структура проекта

```
src/
  App.jsx                  # Роуты + ToastContext
  index.css                # Tailwind + Google Fonts (Inter)
  main.jsx                 # BrowserRouter

  components/
    Layout.jsx             # Sidebar + Navbar + main (адаптивный)
    Sidebar.jsx            # Навигация + список сайтов с кампаниями
    Navbar.jsx             # Тёмный хедер, уведомления, аватар
    Toast.jsx              # Уведомление (success/error/info/warning)
    ToastContainer.jsx     # Fixed bottom-right контейнер

  hooks/
    useToast.js            # addToast / removeToast

  pages/
    Welcome.jsx            # Лендинг: ввод URL или вход через Яндекс
    OAuthCallback.jsx      # Экран авторизации Яндекс (анимация → редирект)
    Analyzing.jsx          # Загрузка анализа (тёмный фон, шаги с анимацией)
    AddSite.jsx            # 4-шаговый wizard добавления сайта
    Dashboard.jsx          # Мои сайты: карточки + график + суммарные метрики
    Results.jsx            # Результаты анализа: табы (ключи / минуса / креативы)
    CampaignDetail.jsx     # Детали кампании: объявления, ключи, настройки
    CampaignCreate.jsx     # (устаревший, заменён на AddSite)
    WeeklyReport.jsx       # Еженедельный отчёт: обзор → по сайтам → аудит → рекомендации
    Hypotheses.jsx         # ИИ гипотезы: тексты + изображения для РСЯ
    Settings.jsx           # Настройки: аккаунт, нейросети, расписание
    Credentials.jsx        # Локальные API ключи (localStorage, не коммитить)
    NotFound.jsx           # 404
```
